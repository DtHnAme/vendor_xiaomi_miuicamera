# MiuiCamera
## Getting Started :
### Cloning :
- Clone this repo in vendor/xiaomi/miuicamera in your working directory by :
```
git clone -b <branch-name> https://github.com/DtHnAme/android_vendor_xiaomi_miuicamera vendor/xiaomi/miuicamera
```

- Done, continue building your ROM as you do normally.
